Trabalhando com Cores em CSS

How to select the colors:

The developer has different ways to select the colors:

  • by name
  • by rgb code
  • by hexadecimal code
  • select the color using the mouse on the palette

How to use color in your website:

The general rule says that a website uses from 3 to 5 colors more than the white (background color) and black (text).

Cores complementares = cores mais contrastantes.

Cores análogas = cores irmãs.

A regra geral de design sugere que para criar uma paleta, de 4 cores, harmonica, vc deve utilizar 3 cores análogas e uma cor complementar.
