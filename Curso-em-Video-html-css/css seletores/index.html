Trabajando con Selectores

Principales tipos de selectores:

  • Selector por Tags: h1, h2, p, div, ul, li, etc.
  • Selector Universal: se representa con '*'. Normalmente es utilizado para 'zerar' las configuraciones padrón del navegador.
  • Selector por 'id' o 'class': El nombre del selector 'id' no se puede repetir en cada página, o sea, puedo tener varios 'ids' en una página, pero cada uno debe tener un nombre distinto. Mientras el selector 'class' si que se puede repetir y contener el mismo nombre en una página.
