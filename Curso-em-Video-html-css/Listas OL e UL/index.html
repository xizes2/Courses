Listas Ordenadas

Para ordenar uma lista vc deve usar a tag 'ol' (ordered list) e depois a tag 'li' para cada item da lista. Vc ainda pode usar os parâmetros 'type', que determina o tipo de 'tick' da lista e também o parâmetro 'start', que determina o 'tick' inicial.

Listas Não Ordenadas

A tag para fazer listas não ordenadas é 'ul' (unordered list) e cada item dentro da lista deve usar a tag 'li'.
