CSS - Trabalhando com estilos

Há 3 formas de trabalhar o estilo das páginas em html: inline, interno e externo. Abaixo veremos as características de cada um deles.

Estilo Inline

O estilo inline deve ser aplicado a cada tag do código, ou seja, se vc tiver 10 paragrafos em uma página, terá que repetir a confirguração de estilo em cada um desses parágrafos.

Essa maneira de estilizar a página é mais utilizada quando vc quer estilizar uma tag em particular.

Ela é a forma de estilização mais forte. Ou seja, se estivermos usando uma das duas outras formas na página, mas uma das tags tiver uma estilização inline, é essa que vai predominar.

Estilo local ou interno

Nessa forma de estilo vc configura o estilo de toda a página em um bloco de código que vem antes do 'body' da página. Na verdade vc vai determinar o estilo de cada tipo de tag que vc usar naquela página.

Por exemplo:

  • vc vai configurar as tags 'h1' para terem a cor vermelha e ficarem alinhadas à direita
  • as tags 'p' terão cor verde e tamanho 14px
  • as tags 'a' abrirão em uma nova aba.

Estilo externo

No estilo externo vc vai configurar um arquivo 'style.css' contendo as configurações de todas as tags que vc quiser e vai aplicar esse arquivo à todas as páginas que desejar.

Para aplicar esse arquivo à página vc deve usar a tag 'link' abaixo do 'title' e referenciar o caminho do arquivo no parâmetro 'href'

Essa é a maneira mais usual para montar o esquema base de estilização do seu site já que vc só precisa criar um arquivo e aplicá-lo à todas as páginas ao invés de fazer isso linha por linha (inline) ou página por página (local).
