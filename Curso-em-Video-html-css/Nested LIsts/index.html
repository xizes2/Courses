Listas Ordenadas

Para ordenar uma lista vc deve usar a tag 'ol' (ordered list) e depois a tag 'li' para cada item da lista. Vc ainda pode usar os parâmetros 'type', que determina o tipo de 'tick' da lista e também o parâmetro 'start', que determina o 'tick' inicial, apenas para ticks numéricos.

Listas Não Ordenadas

A tag para fazer listas não ordenadas é 'ul' (unordered list) e cada item dentro da lista deve usar a tag 'li'.

Nested Lists

Podemos ter listas dentro de listas e misturar os diferentes tipos de listas.

  1. Lista Principal
    1. sub lista
    2. sub lista
    3. sub lista
  2. Lista Principal 2
    1. sub list
    2. sub list
    3. sub list
  3. Lista Principal 3
    • sub lista (sem ordem)
    • sub lista (sem ordem)
    • Sub lista (sem ordem)
