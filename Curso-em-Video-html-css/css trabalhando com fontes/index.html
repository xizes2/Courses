Trabajando con Fuentes

Parámetros de Fuentes

  • font-family: Selecciona la familia de fuente. Si el usuario no tiene la primera fuente instalada en su ordenador, el navegador pasa a 2a, si no tiene la 2a, pasa a la 3a, etc.
  • font-size: Selecciona el tamaño de la fuente. Puedes utilizar varias unidades distintas, como cm, px, pt, etc.
  • font-style: Determina el estilo de la fuente.
  • font-variant: Tiene que ver con la estilización de la fuente. Por ej: small-caps transforma todas las letras minusculas en maiuscualas, pero con el tamaño de minusculas.
  • line-variant: Se refiere al espaciamento de las líneas.
  • text-decoration: Para subliñar o tachar el texto.
  • text-ident: Sirve para hacer el espaciamento inicial del texto.

Hay distintas maneras de hacer la estilización de las fuentes de la página.

Yo puedo especificar todo de manera separada, por si quiero algo muy especifico o si quiero estilizar muchos aspectos distintos de la fuente.

Pero tambien puedo simplificar el proceso utilizando solamente el parámetro 'font' y algunas características básicas. En este caso es importante notar que tendrás que usar al menos 2 características para que funcione.
