Principais formatações de texto:

Negrito / Destaque

Nessa frase existe um termo em negrito usando a tag 'b' (tag não semantica)

Nessa frase existe um termo destacado usando a tag 'strong' (tag semantica)

Visualmente as duas tags terão o mesmo efeito no texto, porém, a tag 'strong' dá sentido de destaque, ou seja, além de alterar a forma (negrito), ela traz destaque ao texto.

Itálico / Ênfase

Nesta frase existe um termo em itálico usando a tag 'i' (não semantica, apenas da forma ao texto)

Nesta frase existe um termo com ênfase usando a tag 'em' (tag semantica, dá sentido ao termo enfatizado)

Marcação de Texto

Também podemos criar um texto marcado usando a tag 'mark'

Para alterar a cor do marca texto eu tenho que usar css dentro da tag 'mark'

Texto Deletado

Para mostrar um texto e dar a ideia de que ele foi excluído, envelopamos o texto usando a tag 'del'. O texto aparecerá riscado

Texto Inserido

Para dar impressão de que o texto foi inserido, envelopamos o mesmo com a tag 'ins'. O texto aparecerá sublinhado.

Texto Sobrescrito

Para 'elevar' um texto a outra variante temos que usar a tag 'sup'. Dessa maneira o numero 2 na expressão x2 + 3, ficará na posição elevada sobre o x. Assim: x2+3

Texto Subscrito

Da mesma maneira podemos subscrever algum valor dentro de um texto, como por exemplo em H2O, usando a tag 'sub'. Dessa maneira: H2O
