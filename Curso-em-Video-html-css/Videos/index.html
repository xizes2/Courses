Trabalhando com Videos

Opção 1 - Hospedando video diretamente no seu servidor (não recomendado)

O procedimento para inserir video é o mesmo para audio, mas usando a tag 'video'. Porém, não é recomendado hospedar os videos diretamente no seu servidor já que os arquivos são muito grandes.

Opção 2 - Usando um link para seu canal no youtube ou vimeo (recomendado)

A opção mais recomendada é linkar à serviços de streaming de videos, como youtube ou vimeo.
