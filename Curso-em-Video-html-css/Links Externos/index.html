Usando Links Externos

Para criar um link para um site externo no seu texto vc deve envelopar a palavra ou texto com a tag 'a'.

Teste 1:

Vc pode acessar o meu repositório público no github. (Esse link vai acessar o site do link na mesma aba)

Se vc quiser abrir uma segunda aba para acessar o link (RECOMENDADO) vc deve usar os parâmetros 'target = _blank', para que a aba onde será aberto o link esteja em branco antes de carregar o site. E também o parâmetro 'rel = external' para abrir o link em uma nova aba.

Teste 2 - Vamos provar a segunda opção:

Vc pode acessar o meu repositório público no github. (Esse link abrirá em uma nova aba)
