Inserindo Audio no Site

Para inserir um audio vc deve usar a tag 'audio' e dentro dela utilizar a tag 'source src' e selecionar o caminho do arquivo. Para mostrar o controle do audio vc deve usar o parametro 'controls' dentro da tag 'audio'.

Assim como para fotos, audios (e qualquer tipo de midia) também possuem a questão dos direitos autorais. Uma boa fonte para baixar arquivos de audio autorizados pelos criadores é a biblioteca de audios do youtube. Lá vc pode escolher os audios livres, com e sem necessidade de atribuição ao criador.
